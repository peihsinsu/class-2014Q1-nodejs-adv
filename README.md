Node.js進階課程
====

## 目的：

介紹與推廣雲端之重要開發工具Node.js如何輕鬆開發與介接REST服務

## 參加人員限制：

已有Node.js基礎

## 大綱：

* 雲端API服務介接
	* request模組介紹
	* 解析google oauth與google api
* SocketIO的應用
	* Node.js的SocketIO套件介紹
	* Server Socket
	* Client Socket
	* SocketIO應用範例 - Online Chat
* 輕輕鬆鬆學NoSQL
	* 什麼是NoSQL
	* CouchDB連接程式實作
	* Map範例與使用時機
	* Reduce範例與使用時機
	* 有名的NoSQL使用範例介紹
